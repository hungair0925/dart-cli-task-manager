1. プロジェクト目的
- Dartの基本文法を、CLI開発(タスク管理ツール)を通じて習得

2. タスク
  - 識別子
  - タイトル
    - 1文字以上30文字以内
    - 空文字は許容しない
  - 完了期限
  - 完了状態
    - 未完了/完了
  - 作成日時

2. 機能
- [ ] タスク追加
  - 名前と期限を指定して新しいタスクを作成する。
- [ ] タスク一覧表示
  - 登録されている全てのタスクを期限順/登録順で表示する。
- [ ] タスクの完了切り替え
  - ID（またはインデックス）を指定して、完了/未完了を反転させる。
- [ ] タスク削除
  - 指定したタスクをリストから完全に削除する。
- [ ] データの永続化
  - アプリ終了時にファイルへ保存し、起動時に自動読み込みを行う。
